class MatchedMediaItem : MediaItem
Content copied to clipboard

MatchedMediaItem represents metadata that has been matched against a Catalog.

Extra information is presented that can only be generated from a match. The properties provided here are ephemeral and can differ each time there is a match of the Signature that this object represents.
